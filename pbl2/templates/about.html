Project Overview

The Page Replacement Simulator is a web-based tool designed to simulate various page replacement algorithms that are used in operating systems to manage memory. This tool provides an interactive platform to visualize and compare the performance of different algorithms in a memory management context.

Algorithms Implemented

The simulator includes the following algorithms:

  • FIFO (First In First Out): Replaces the page that has been in memory the longest.
  • LRU (Least Recently Used): Replaces the page that has not been used for the longest period.
  • Optimal: Replaces the page that will not be used for the longest period in the future.
  • Clock: Uses a circular approach to replace pages based on a clock-like mechanism.

Technologies Used

This project was developed using Python for the backend, where the algorithms are implemented and executed. The front-end is built with HTML, CSS, and JavaScript for seamless user interaction and a clean, modern interface.

Educational Purpose

Through this simulator, users can gain a deeper understanding of memory management and page replacement techniques. This project serves as both an educational tool and a practical application for studying algorithms.
